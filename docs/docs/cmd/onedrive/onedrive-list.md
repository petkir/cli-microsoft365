# onedrive list

Retrieves a list of OneDrive sites

## Usage

```sh
m365 onedrive list [options]
```

## Options

--8<-- "docs/cmd/\_global.md"

## Examples

Retrieves a list of OneDrive sites from the tenant.

```sh
m365 onedrive list
```
